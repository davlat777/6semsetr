# ЛК3 19.02.2024 Дефекты

**Отчет о дефекте** - документ, содержащий отчет о любом недостатке в компоненте или системе, который может привести компонент или систему к невозможности выполнить требуемую функцию.

![image](https://github.com/davlat777/6semsetr/assets/113089483/d928b002-e607-4788-aa0c-c2458a9e1c23)

Отчет о дефекте:

идентификатор

1.краткое описание: характер проблемы, границы дефекта, полседствие дефекта

2.подробное описание: ожидаеые и фактические результаты

3.влияние: критичность и приоритет

Причины почему дефекты не устраняют:

1.нехватка времени

2.дефект вовсе не дефект

3.устранять неисправность слишком рискованно

4.это того не стоит

5.неэффективный отчет о дефектах

**Упражнение 1:** Заголовок: Некорректные результаты сложения в калькуляторе Windows.

Описание: При проведении тестирования калькулятора в операционной системе Windows были обнаружены некорректные результаты при выполнении сложения четных чисел

Шаги вопроизведения:

Запустить калькулятор Windows. Выполнить операцию сложения четных чисел. Посмотреть результат сложения. Ожидаемый результат: Калькулятор должен корректно выполнять арифметические операции и возвращать правильные результаты сложения.

Наблюдаемый результат: Калькулятор выдает неправильные результаты сложения для четных чисел.

Влияние: Критичное.

Тестовый набор:

1.Тестовые сценарии

3.Правила генераци

1.Покрытие операторов

2.Узлов ветвления
